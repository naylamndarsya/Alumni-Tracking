# Sistem Pelacakan Alumni

## Deskripsi

Sistem Pelacakan Alumni adalah aplikasi web yang digunakan untuk mencatat dan memantau data alumni suatu program studi atau sekolah.
Melalui sistem ini, pengguna dapat menambahkan data alumni, memperbarui status pekerjaan alumni, mencari data alumni, serta mengelola data alumni dengan fitur edit dan hapus.


## Fitur Sistem

* Menambahkan data alumni
* Mengedit data alumni
* Menghapus data alumni
* Memperbarui status pekerjaan alumni
* Pencarian data alumni
* Penyimpanan data menggunakan database Supabase

## Teknologi yang Digunakan

* HTML
* Tailwind CSS
* JavaScript
* Supabase (Database)
* GitHub Pages (Hosting)

## Pengujian Sistem

| No | Fitur yang Diuji     | Langkah Pengujian                       | Hasil    |
| -- | -------------------- | --------------------------------------- | -------- |
| 1  | Tambah Alumni        | Mengisi form dan klik tombol tambah     | Berhasil |
| 2  | Edit Alumni          | Klik tombol edit lalu ubah data         | Berhasil |
| 3  | Hapus Alumni         | Klik tombol hapus pada data alumni      | Berhasil |
| 4  | Update Status Alumni | Mengisi data pekerjaan alumni           | Berhasil |
| 5  | Pencarian Alumni     | Mengetik nama alumni di kolom pencarian | Berhasil |
| 6  | Koneksi Database     | Data tersimpan di Supabase              | Berhasil |

## Kesimpulan

Aplikasi Sistem Pelacakan Alumni berhasil dibuat dan dapat digunakan untuk mengelola data alumni secara sederhana. Sistem ini mampu melakukan operasi CRUD (Create, Read, Update, Delete) serta menyimpan data secara online menggunakan database Supabase.
